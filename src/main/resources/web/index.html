RDV-HTTP

  • / help page
  • /stat server stats
  • /sessions show session list
  • /ping?dsn=host:port show list of endpoints and channels
  • /session/start?dsn=$dsn&limit=1000&t=15 start data stream session to target DSN (format: host:port/endpoint/channel) with limited length of buffer and session timeout (ms)
  • /session/stop?sid=$sid stop session with input sid
  • /session/$sid flush data buffer to JSON format
